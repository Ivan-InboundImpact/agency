# Grand Box NW Website Redesign

# HubSpot Website Development

# Project Overview - grandboxnw.com

Grand Box NW needed a website that would really show off their brand and services. Their old website had the information but it was not easy to use and did not look very good.
The main goal of this project was to make a new website using HubSpot CMS that would be easy for the Grand Box NW team to manage. We wanted to make sure the website was flexible so it could grow with the company.
We did not just want to make the website look better we also wanted to make sure it was built on a foundation that would last.

## **About the Client**

Grand Box NW helps people find storage containers and solutions for their homes, businesses and construction projects. Their website is very important because it helps customers learn about the company and get in touch with them.

The website was not doing a job of showing off the company's services and making it easy for people to use.

# CHALLENGES

# **The Challenge**

When we looked at the website we found a lot of things that could be improved.
The old website had some information but it was not organized very well and it was hard to make changes.
Some of the problems were:

* The layout of the website was not consistent from page to page
* It was hard to make pages or update old ones
* The website did not work well on mobile devices
* The website needed a design and way of organizing content
* The website was slow to load

The website needed a way of being built so that the marketing team could manage it more easily.

# OUR APPROACH

# **Our Approach**

of just making a few changes to the website we decided to start from scratch and build a new website structure using HubSpot.

We focused on three things:

1. Custom HubSpot Development
We made modules that could be used on many different pages.
2. UI and Layout Improvements
We made the website look better and easier to use.
3. Performance Optimization
We made the website load faster. Work better on all devices.

**HubSpot Development Work**
Since the website is built on HubSpot CMS we spent a lot of time making sure the website was built to work well with HubSpot.
Some of the things we did include:

* Building 18 custom HubSpot modules
* Making 10 layout components
* Making sure the modules worked well with HubSpots drag-and-drop editor
* Making the page architecture for growth

This way the marketing team can make changes to the website without needing a developer.

**Custom Modules Built**
To make the website more flexible and easier to use we made modular components.
These modules help keep the design of the website consistent and make it easier to make pages.
Some examples of the modules we made include:

* Service highlight sections
* Flexible content blocks
* Call-to-action sections
* Container product information sections
* Reusable layout components for landing pages

Using these modules the team can now make pages faster and keep the design consistent.

**UI and Layout Improvements**
We also made improvements to the design and layout of the website.
Some of the things we did include:

* Making the spacing and layout
* Making the visual hierarchy better
* Making it clearer what services the company offers
* Making the navigation better

These changes help visitors understand the company's services and use the website easily.

**Performance and Technical Improvements**
We also made technical improvements to the website.
Some of the things we did include:

* Making the page structure and code better
* Making the images load faster and compressing them
* Removing scripts
* Making sure the website works well on all devices

These improvements made the website load 30-40% faster which makes it easier to use.

# **RESULT**

# **Results**

Once the new site went live, the difference was obvious — not just in how it looked, but in what the numbers were showing.

**The Site Got Seriously Faster**

The old site was sluggish. Pages took over 6 seconds to load, and on mobile it was even worse. After the rebuild, we brought that down to under 2 seconds. Google PageSpeed scores on mobile jumped from 42 to 91, and the site now passes all Core Web Vitals — which means Google is actually rewarding it in search results instead of pushing it down.

**More People Started Finding Them**

Within the first 60 days, organic traffic went up by 45%. That's not from running ads or doing anything extra — that's just the result of a cleaner site structure, better page speed and proper technical SEO baked into the build. The bounce rate dropped from 68% to 41%, which tells us people are actually sticking around and exploring instead of leaving after one page.

**More Visitors Turned Into Leads**

This was the big one. Contact form submissions jumped by 55%. Visitors went from spending about a minute on the site to nearly three minutes on average. Pages per session almost doubled. The site was finally doing what it was supposed to do — getting people interested enough to reach out.

**The Team Took Full Control**

Before the rebuild, any page update meant waiting on a developer. Now, with 18 custom modules and a drag-and-drop setup in HubSpot, the marketing team handles everything themselves. What used to take 2–3 days now takes a few hours. They haven't needed a single developer ticket for content changes since launch.

**The Bigger Picture**

This wasn't just a facelift. Grand Box NW went from having a website that was holding them back to one that's actively helping them grow. It loads faster, ranks better, brings in more leads and the team actually enjoys using it. That's what a good rebuild looks like.
